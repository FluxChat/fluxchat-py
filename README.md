# PyChat

## Installation

```bash
./bin/setup.sh
```

To setup another node, use environment variables.

```bash
. .venv/bin/activate
export PYCHAT_CONFIG=var/config2.json
export PYCHAT_PORT=25002
export PYCHAT_DATA_DIR=var/data2
./bin/setup.sh
```
